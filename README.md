# MAVha Todo REST API

## Requirements
* NodeJS 10.15.2
* Microsoft SQL Server

## Setup
#### Dependencies
```
 npm install
```
#### Database creation
```
npm run db:creates
```
#### Database migration
```
npm run db:migrate
```
#### Database seeds
```
npm run db:seeds
```

## Running

#### Tests
```
npm run tests
```
#### Development
```
npm run dev
```
#### Production
```
npm run prod
```
